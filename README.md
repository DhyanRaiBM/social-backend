# UpQuarium Backend

Backend repository for UpQuarium, a social media application developed with MERN stack.

## Features

- User authentication (JWT-based)
- CRUD operations for posts
- User profiles and activity feed

## Technologies Used

- **MongoDB** 
- **Express.js** 
- **React**
- **Redux Toolkit**
- **Tailwind CSS**
- **Node.js** 
- **Mongoose** 
- **JWT**
